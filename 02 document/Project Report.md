# Project Report

# Vrinda Store Sales Analysis

## 1. Project Overview

The Vrinda Store Sales Analysis is an interactive Microsoft Excel dashboard designed to analyze sales performance using historical order data. The dashboard provides insights into sales, orders, customer demographics, order status, sales channels, and regional performance through interactive visualizations.

This project demonstrates how Excel can be used to clean, analyze, and visualize business data for decision-making.

---

## 2. Objectives

* Analyze overall sales and order performance.
* Compare sales based on customer gender and age groups.
* Identify the top-performing states.
* Evaluate sales across different sales channels.
* Analyze order status and fulfillment trends.
* Create an interactive dashboard for business reporting.

---

## 3. Dataset

The dataset contains sales transaction records of Vrinda Store, including:

* Order ID
* Customer Gender
* Customer Age
* State
* Sales Amount
* Order Status
* Sales Channel
* Product Category

The data was cleaned, organized, and analyzed within the same Excel workbook before building the dashboard.

---

## 4. Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Excel Formulas
* Conditional Formatting

---

## 5. Dashboard Features

The dashboard includes:

* Interactive slicers for filtering data
* Sales and order KPIs
* Sales by gender
* Age vs. gender analysis
* Top 10 states by sales
* Sales by channel
* Order status distribution
* Monthly sales and order trends

## 7. Conclusion

The Vrinda Store Sales Analysis dashboard provides a clear overview of sales performance and customer behavior. It helps identify key business trends, monitor sales across different channels and regions, and supports data-driven decision-making through an interactive Excel dashboard.
