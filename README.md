# 📊 Vrinda Store Sales Analysis

An interactive **Microsoft Excel dashboard** that analyzes Vrinda Store's sales performance using historical order data. The dashboard provides valuable insights into sales trends, customer demographics, sales channels, order status, and regional performance through interactive charts and filters.

---

## 📌 Project Overview

This project transforms raw sales data into an interactive dashboard that enables users to explore business performance and customer behavior. The dashboard helps identify key trends and supports data-driven decision-making using Excel's analytical features.

---

## 🎯 Objectives

* Analyze overall sales and order performance.
* Compare sales by customer gender and age group.
* Identify the top-performing states.
* Evaluate sales across different channels.
* Analyze order status and fulfillment trends.
* Create an interactive business dashboard.

## 📈 Key Insights

* Women contributed more sales than men.
* Adult customers generated the highest number of orders.
* Amazon was the leading sales channel.
* Maharashtra, Karnataka, and Uttar Pradesh were among the top-performing states.
* Most orders were successfully delivered, indicating a strong fulfillment rate.
